# MyP_2024-1
## This is the official repository of Equipo's MyP class projects for semester 2024-1.
These projects showcase some case uses for the various pattern designs out there.

### All projects were made in Java using the specified pattern designs.
### A quick overview of the projects (Click on each one to see more details):
+ [Práctica 1: An in-terminal game (it is rigged)](/practica1)
+ [Práctica 2: A cooking "bot" that uses state to give you an experience. (terminal)](practica2)
+ [Práctica 3: An in-terminal army builder.](practica3)
+ [Práctica 4:A spaceship creator.](practica4)
+ [Proyecto 1: A shop with discounts.](/proyecto1)
+ [Proyecto 2:A text editor based on servers with a javaFX interface.](/proyecto2)
